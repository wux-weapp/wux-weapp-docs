# 更新日志

`Wux Weapp` 严格遵循 [Semantic Versioning 2.0.0](http://semver.org/lang/zh-CN/) 语义化版本规范。

- 修订版本号（0.0.x）：向下兼容的 bugfix 修复。
- 次版本号（0.x.0）：含有新特性的向下兼容的版本。
- 主版本号（x.0.0）：含有破坏性更新和新特性，不向下兼容的版本。
