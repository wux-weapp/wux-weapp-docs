# Wux Weapp

`Wux Weapp` 是一套组件化、可复用、易扩展的微信小程序 UI 组件库。

## 特性

- `UI` 样式可配置，拓展灵活，轻松适应不同的设计风格
- `80+` 丰富的组件，能够满足移动端开发的基本需求

## 支持环境

- `v3.6.0+` 需要依赖小程序基础库 `2.6.6` 以上版本
- `v3.4.0+` 需要依赖小程序基础库 `2.2.3` 以上版本
- `v3.0.0 ~ v3.3.2` 需要依赖小程序基础库 `1.6.3` 以上版本
